# MyCash — План приложения (brief)

**Дата:** 08.04.2026
**Формат:** Telegram Mini App
**Дизайн:** Светлый Apple-стиль, минимализм
**Навигация:** Один экран с прокруткой (без вкладок)

---

## Экран: Главная (один экран, прокрутка)

### Блок 1 — Баланс (всегда вверху)
| Элемент | Описание |
|---------|----------|
| Общий баланс | Крупная цифра "53 000 ₽" |
| 💳 Карта | Баланс карты: "45 000 ₽" |
| 💵 Наличка | Баланс наличных: "8 000 ₽" |

### Блок 2 — Период
| Элемент | Описание |
|---------|----------|
| Кнопки периода | Сегодня / Неделя / Месяц |
| Доходы за период | "+50 000 ₽" зелёным |
| Расходы за период | "-12 000 ₽" красным |

### Блок 3 — Дашборд (круговая диаграмма)
| Элемент | Описание |
|---------|----------|
| Donut-диаграмма | Расходы по категориям |
| Легенда | Топ-5 категорий с суммами и % |
| Скрывается | Если нет расходов за период |

### Блок 4 — Список операций
| Элемент | Описание |
|---------|----------|
| Операция | Иконка + Категория + Комментарий + Кошелёк + Сумма |
| Нажатие | Открывает редактирование или удаление |
| Пустое состояние | "Пока нет операций" + призыв добавить |

### Блок 5 — Кнопки внизу (плавающие)
| Элемент | Описание |
|---------|----------|
| 🎤 | Голосовой ввод |
| + Новая операция | Открывает форму ввода |

---

## Модальное окно: Новая операция

| Элемент | Описание |
|---------|----------|
| Переключатель | Расход / Доход / **Перевод** |
| Сумма | Крупное поле, цифровая клавиатура |
| Кошелёк | 💳 Карта / 💵 Наличка (для расхода/дохода) |
| Кошелёк откуда → куда | Только для перевода: "С карты → В наличку" |
| Категория | Кнопки-чипы (8-10 штук) |
| Дата | Выбор даты (по умолчанию сегодня) |
| Комментарий | Текстовое поле (необязательно) |
| Кнопка "Сохранить" | Сохраняет и закрывает |

**Категории расходов:** Продукты, Кафе, Транспорт, ЖКХ, Связь, Одежда, Здоровье, Развлечения, Подписки, Прочее

**Категории доходов:** Зарплата, Подработка, Кэшбек, Подарок, Возврат, Прочее

**Перевод:** Без категории — только сумма и направление (Карта → Наличка или Наличка → Карта)

---

## Голосовой ввод

| Этап | Что происходит |
|------|---------------|
| 1 | Нажал 🎤 → красный пульсирующий экран "Слушаю..." |
| 2 | Сказал: "Расход пять тысяч продукты" |
| 3 | Парсер разбирает: тип, сумма, категория |
| 4 | Подтверждение: "Расход 5 000 ₽ — Продукты. Записать?" [Да] [Изменить] |
| 5 | "Да" → записано + вибрация. "Изменить" → открывается форма с предзаполненными данными |

---

## Онбординг (первый запуск)

| Этап | Что видит пользователь |
|------|----------------------|
| 1 | Приложение открывается с демо-данными (17 операций) |
| 2 | Баннер сверху: "Это примеры. Нажми чтобы очистить и начать свой учёт" |
| 3 | Кнопка "🗑 Очистить и начать" → удаляет демо-данные |
| 4 | После очистки: "Сколько у тебя на карте?" → поле ввода → "А наличных?" → поле ввода |
| 5 | Установлены начальные остатки → пустое приложение готово к работе |

---

## Действия пользователя

| Действие | Как |
|----------|-----|
| Добавить операцию | Кнопка "+ Новая операция" или 🎤 |
| Удалить операцию | Нажать на операцию → подтверждение |
| Редактировать операцию | Нажать на операцию → форма с данными |
| Перевод между кошельками | "+ Новая операция" → вкладка "Перевод" |
| Сменить период | Кнопки Сегодня/Неделя/Месяц |
| Очистить демо-данные | Баннер сверху → "Очистить и начать" |

---

## Чего НЕ будет в первой версии

- Бюджеты и лимиты по категориям
- Повторяющиеся операции
- Экспорт данных (CSV/Excel)
- Тёмная тема
- Несколько валют
- Напоминания (нужен VPS)
- Настройки (свои категории, время напоминаний)
- Совместный доступ (семья)

---

## Общие требования

- Минимальный размер кнопок: 44x44px
- Шрифт не менее 14px
- Крупные кнопки категорий
- Баланс виден сразу при открытии
- Ввод операции — максимум 3 тапа
- HapticFeedback при сохранении
- Подтверждение перед удалением (tg.showConfirm)
- Данные сохраняются в localStorage + CloudStorage
