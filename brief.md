# MyCash — План приложения (brief)

**Дата:** 08.04.2026
**Формат:** Telegram Mini App
**Дизайн:** Светлый Apple-стиль, минимализм
**Навигация:** Нижнее меню (таб-бар) + экраны с прокруткой. Даже в MVP — приучать пользователей к навигации для будущего Pro
**Цена:** Freemium — бесплатный базовый учёт, Pro 149 ₽/мес

---

## Общие требования
- Светлый Apple-дизайн, минимализм
- Русский язык интерфейса
- Крупные кнопки (минимум 44px), шрифт не менее 14px
- **Ввод расхода — максимум 2 тапа**
- HapticFeedback при каждом действии
- MainButton и BackButton Telegram
- tg.showConfirm() вместо confirm()
- Данные на сервере (FastAPI + SQLite), не в localStorage

---

## Экран: Главная (один экран, прокрутка)

### Блок 1 — Баланс (вверху)
| Элемент | Описание |
|---------|----------|
| Общий баланс | Крупная цифра "53 000 ₽" |
| 💳 Карта | "45 000 ₽" |
| 💵 Наличка | "8 000 ₽" |

### Блок 2 — Период + доходы/расходы
| Элемент | Описание |
|---------|----------|
| Сегодня / Неделя / Месяц | Переключатель |
| Доходы | "+50 000 ₽" зелёным |
| Расходы | "-12 000 ₽" красным |

### Блок 3 — Дашборд
| Элемент | Описание |
|---------|----------|
| Donut-диаграмма | Расходы по категориям (интерактивная — тап на сегмент = список операций) |
| Легенда | Топ-5 категорий с суммами и % |

### Блок 4 — Список операций
| Элемент | Описание |
|---------|----------|
| Операция | Иконка + Категория + Комментарий + Кошелёк + Сумма |
| Тап | Открывает редактирование |
| Свайп влево | Удаление с подтверждением (tg.showConfirm) |

### Блок 5 — Плавающие кнопки внизу
| Элемент | Описание |
|---------|----------|
| 🎤 | Голосовой ввод (бесплатно) |
| + | Форма ввода операции |

## Модальное окно: Новая операция (быстрый ввод — 2 тапа)

**По умолчанию — режим быстрого ввода:**
| Элемент | Описание |
|---------|----------|
| Поле суммы | Крупное, цифровая клавиатура, автофокус |
| Быстрые категории | 8-10 круглых иконок — тап = сохранено! |
| Кошелёк | 💳/💵 переключатель (по умолчанию последний использованный) |

Дата = сегодня (автоматически). Тип = расход (по умолчанию). Комментарий = пусто.

**Расширенный режим (кнопка "Подробнее"):**
| Элемент | Описание |
|---------|----------|
| Переключатель | Расход / Доход / Перевод |
| Дата | Выбор (если нужно задним числом) |
| Комментарий | Текстовое поле |
| Кошелёк откуда → куда | Только для перевода |

**Категории расходов:** Продукты, Кафе, Транспорт, ЖКХ, Связь, Одежда, Здоровье, Развлечения, Подписки, Прочее

**Категории доходов:** Зарплата, Подработка, Кэшбек, Подарок, Возврат, Прочее

**Перевод:** Без категории — только сумма и направление (Карта → Наличка или Наличка → Карта)

## Голосовой ввод (бесплатно)

| Этап | Что происходит |
|------|---------------|
| 1 | Нажал 🎤 → красный пульсирующий экран |
| 2 | Сказал: "Расход пять тысяч продукты" |
| 3 | Подтверждение inline: предзаполненная карточка [✅ Записать] [✏️ Изменить] |
| 4 | Записать → сохранено + вибрация. Изменить → форма с данными |
| Фоллбек | Если голос не сработал — сразу ручной ввод без ошибок |

## Онбординг (первый запуск)

| Этап | Что видит |
|------|----------|
| 1 | Демо-данные (17 операций, реалистичные суммы) |

| 6 | Подсказка: "Запишите первый расход" с указателем на кнопку + |

## Действия пользователя

| Действие | Как |
|----------|-----|
| Добавить расход | + → сумма → тап категория → готово (2 тапа) |
| Добавить доход | + → Подробнее → Доход → сумма → категория |
| Перевод | + → Подробнее → Перевод → сумма → откуда/куда |
| Голосом (Pro) | 🎤 → сказал → подтвердил |
| Редактировать | Тап на операцию → форма с данными |
| Удалить | Свайп влево → подтверждение |
| Сменить период | Кнопки Сегодня/Неделя/Месяц |
| Очистить демо | Баннер → "Очистить и начать" |
